# Getting Started

This section guides you through installing charip-lsp and configuring your editor.

## Prerequisites

- A MyST/Sphinx documentation project
- One of the supported editors:
  - Neovim (0.8+)
  - VS Code
  - Any editor with LSP support

## Quick Start

1. **Install** the charip-lsp binary
2. **Configure** your editor to use it
3. **Open** a `.md` file in your documentation project

The LSP will automatically index your vault and provide intelligent features.
