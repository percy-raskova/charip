# Contributing to charip-lsp

Thank you for your interest in contributing to charip-lsp. This section provides everything you need to get started.

## Ways to Contribute

### Report Issues

Found a bug or have a feature request? Open an issue on GitHub with:
- Steps to reproduce (for bugs)
- Expected vs actual behavior
- Your environment (editor, OS)

### Submit Code

We welcome pull requests for:
- Bug fixes
- New features
- Documentation improvements
- Test coverage

### Improve Documentation

Documentation contributions are valuable:
- Fix typos and unclear explanations
- Add examples
- Improve tutorials

## Code of Conduct

Be respectful and constructive. We're building tools for documentation writers—let's communicate like we'd want our docs to read.

## Getting Help

If you're stuck:
1. Check existing issues for similar problems
2. Open a new issue with details
3. Be patient—maintainers are volunteers
